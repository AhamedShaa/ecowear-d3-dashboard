# EcoWear Analytics Dashboard

Interactive D3.js dashboard exploring:
- Weather vs product performance
- Customer flow through funnel stages
- Product sales by customer segment
- Monthly product sales trends

## Project Structure
- `/1` Radar Chart
- `/2` Sankey Diagram
- `/3` Stacked Coxcomb Chart
- `/4` Stacked Bar Chart
- `dashboard.html` combined dashboard

## Features
- Interactive tooltips
- Hover highlights
- Legends
- Combined dashboard view

## Datasets
- EcoWear_Product_Weather.csv
- EcoWear_Customer_Flow.csv
- EcoWear_Product_Sales.csv
- EcoWear_Monthly_Sales.csv
